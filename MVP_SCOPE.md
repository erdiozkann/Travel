## MVP Şehir
- CITY: Barcelona

## MVP Özellikleri
1. Auth + dil seçimi
2. Harita (pin + filtre + mini kart)
3. Place/Experience detay sayfası (fiyat range + puan + “local” sinyali)
4. Satın alma (Stripe checkout ile hızlı başlangıç)
5. Post + Check-in (anlık konum + mekan seçimi + foto)
6. Feed + Follow + Profile
7. Review/Rating (sadece satın alan + completed booking)
8. AI Basic (şehir asistanı: 1-3 gün plan, bütçe aralıkları, confidence)

## MVP’de Olmayacaklar
- Tam DM sistemi (sadece booking sonrası business chat opsiyonel)
- Offline harita paketleri
- Çok karmaşık sponsorlu açık artırma (MVP: CPC + puan eşiği)
